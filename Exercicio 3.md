# Exercício 3

* Classe - classe é uma identificação de objetos, é dentro dela que colocamos nossos métodos e atributos, que ira determinar oque o objeto ira fazer.
* Propriedade de uma classe - Propriedades é tudo aquilo que ira compor meu objeto
* Métodos - tudo que o objeto ira fazer, são as funcionalidades de uma classe.
* Instancia é criar um novo Objeto é a representação de uma classe.
* Construtor é um método que é executado quando vou criar um objeto obrigatoriamente o objeto só pode ser criado se passar pelo construtor.
